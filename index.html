HTML

  • HTML is used to create the structure of a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Meta elements (nested within the head element) define metadata such as the character set, description, keywords, author, and viewport.
  • The charset element sets our character encoding.
  • The http-equiv element sets a HTTP header for content.
  • The viewport element sets the default view for out webpage.
  • The non-semantic element div creates divisions on the page.
  • The h1 through h6 elements reresent heading levels. H1 being the largest.
  • The p element represents a paragraph.
  • The ul, ol and li elements represents unordered lists, ordered lists and list items.
  • The img elelment contains info about images displayed on the webpage.
  • Img elements can contain various attributes such as src, alt and dimensions for the image.
  • The `src` attribute defines the location, or source, of the image file.
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image.
  • The br element creates a line of empty space (a line break) between blocks of content.
  • The a element (anchor) creates hyperlinks. It contains the href attribute, which points to the URL for the link.

CSS

  • A margin propery indicates how much space we want around the outside of an element
  • The padding property adds space around the content inside an element
  • The img selector assigns property values to any image in the markeup
  • The box model states that everything in CSS has a box around it
